# Huriwake (ファイル振り分け君)

「Huriwake」は、デスクトップのショートカットにファイルをドラッグ＆ドロップするだけで、あらかじめ設定したフォルダに自動で振り分けてくれる便利ツールです。

## 🔹 主な機能

- **自動振り分け**: 拡張子ごとに移動先フォルダを設定可能。
- **フォルダドラッグ対応**: フォルダをドロップすると、中の設定済み拡張子ファイルのみを抽出して整理。
- **重複ファイル対応**: 移動先に同名ファイルがある場合、「上書き」「変名保存」「キャンセル」を選択可能。
- **リザルト表示**: どのファイルをどこに移動したか、処理後に一覧で確認。
- **フォルダを開く**: 設定画面から直接、移動先フォルダをエクスプローラーで表示。
- **簡単ショートカット作成**: アプリ内からデスクトップへ専用アイコンを作成。

## 🚀 インストール方法

1. **EXEファイルのダウンロード**:
   `dist/huriwake.exe` をダウンロードし、PCの適当な場所（「マイドキュメント」など）に保存してください。
   *※Pythonのインストールは不要です。*

2. **初期設定**:
   `huriwake.exe` をダブルクリックして起動します。

3. **振り分けルール登録**:
   - プルダウンから拡張子（例: `.pdf`）を選択または入力。
   - 「参照」ボタンで、その種類のファイルの移動先フォルダを選択。
   - 「登録」ボタンを押して保存します。

4. **ショートカットの作成**:
   「デスクトップにショートカットを作成」をクリックします。デスクトップに青いフォルダのアイコンが現れます。

## 💡 使い方

- **ファイルを整理する**: 
  デスクトップの「Huriwake」ショートカットに、整理したいファイルをドラッグ＆ドロップしてください。
- **まとめて整理する**: 
  ファイルが混ざったフォルダをショートカットにドロップしてください。設定済みのファイルだけが自動で各所へ飛んでいきます。
- **設定を変更する**: 
  `huriwake.exe` を直接起動すると、いつでも設定画面が開きます。

## 🛠 開発者向け (ソースコードから実行する場合)

### 環境
- Python 3.x
- 依存ライブラリ: `Pillow`, `pyinstaller` (ビルド用)

### 手順
```bash
# 依存ライブラリのインストール
pip install pillow

# 実行
python main.py
```

### ビルド (EXE化)
```bash
python -m PyInstaller --noconsole --onefile --icon=app_icon.ico main.py --name huriwake --noconfirm
```
